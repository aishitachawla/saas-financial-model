# SaaS Financial Model (VC-Level)

This project is a 3-year financial model built in Excel for SaaS startups.

## Features
- Monthly revenue forecasting (36 months)
- Customer growth with churn modeling
- Cost structure (fixed and variable)
- Profitability and cash runway tracking
- Scenario analysis (Base, Best, Worst)
- Unit economics (CAC, LTV, LTV/CAC)

## Use Case
Helps founders and investors evaluate growth, burn rate, and business sustainability.
